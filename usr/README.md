# Term Paper Reader: Piracy & IP Protection

This repository contains a specialized Flutter application built to elegantly present a comprehensive term paper titled **"Piracy and the Protection of Ideas, Concepts, Products, and Enterprises."**

## Overview
The application functions as an interactive document reader, perfectly suited for long-form academic or professional reading. It fulfills the requirement for a structured, multi-chapter term paper including an abstract, detailed analysis sections, and properly formatted references.

## Features
- **Responsive Design:** A beautiful side-navigation drawer for desktop/tablet sizes, and a clean AppBar menu for mobile viewing.
- **Full Academic Structure:** Includes an Abstract, Introduction, detailed analytical chapters (Anatomy of Piracy, Legal Frameworks, Strategic Protection), Conclusion, and 6 verifiable academic/professional References.
- **Distraction-Free Reading:** Carefully chosen typography, line height, and color schemes to reduce eye strain when reading long texts.
- **Cross-Platform:** Runs natively on Web, iOS, Android, macOS, Windows, and Linux.

## How to Run

1. Ensure you have [Flutter](https://flutter.dev) installed on your system.
2. Clone this repository and navigate to the project root.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the application on your preferred device or browser.

## Tech Stack
- **Framework:** Flutter / Dart
- **Architecture:** Standard Stateful/Stateless widget tree with responsive layout builders.
